# 🧠 Brain Tumor Classification

A deep learning project for classifying Brain MRI images into different tumor categories using Convolutional Neural Networks (CNN) and Transfer Learning.

This repository documents the complete workflow from data inspection and preprocessing to CNN and EfficientNet-based model development.

---

## 📌 Project Objective

The objective of this project is to develop a reliable deep learning model capable of classifying brain MRI images into four categories:

- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor

The project focuses on proper data preparation, model training, and performance evaluation using modern deep learning techniques.
